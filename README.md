## How to run the app 
Inside a folder, run:
```bash
$ git clone https://github.com/HMDarkFir3/weather-forecast.git
```
At the root of the project, run:
```bash
npm install  
or 
yarn install
```
After installing the project dependencies, run:
```bash
expo start
```
